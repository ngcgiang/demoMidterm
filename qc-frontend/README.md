# QC Frontend Demo (ReactJS)

This project is a ReactJS UI demo for QCTest workflows.

## Run

1. Ensure backend demo is running at http://localhost:3300
2. Install and start frontend:
   - npm install
   - npm run dev
3. Open:
   - http://localhost:5174

## Optional env

Create .env file:

VITE_API_BASE=http://localhost:3300/api

## Included demo screens

- Dashboard
- Lots
- Workbench (create test, decision, retest, evidence)
- Timeline
- Supplier report
- Return requests
- Audit logs
